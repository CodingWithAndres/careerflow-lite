# CareerFlow Lite

CareerFlow Lite is a lightweight web application designed to help users track job applications in one organized place. The project focuses on simplicity, usability, and fast interaction without requiring accounts or complicated setup.

## Live Demo

🌐 Live Site:  
https://codingwithandres.github.io/careerflow-lite/

## Features

- Add and manage job applications
- Interactive dashboard with application totals
- Filter applications by status
- Search by company or role
- Edit existing applications
- Update application statuses
- Delete applications
- Local storage persistence
- Responsive design for desktop and mobile

## Technologies Used

- HTML
- CSS
- JavaScript
- localStorage
- GitHub Pages

## Project Purpose

The goal of CareerFlow Lite was to create a focused productivity tool for students, early-career professionals, and job seekers who need a simple way to organize their application process.

## Iteration and Improvements

Between Project 01 and the final submission, the project evolved from a mostly static layout into a more interactive application. Improvements included:

- Interactive dashboard filtering
- Inline editing functionality
- Search and filtering
- Status updates
- Improved workflow and usability
- UI polish and responsive behavior

## Usage Instructions

1. Open the live site
2. Add a job application using the form
3. Use dashboard cards or filter buttons to filter applications
4. Search applications by company or role
5. Edit or update statuses directly in the table
6. Data automatically saves in the browser using localStorage
